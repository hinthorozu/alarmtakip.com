### Servislerde kullandığım teknik yapılar
* CQRS (MediatR (Command,Query,Pipeline))
* Domain Driven Design
* IOC (AutoFac)
* Mapping (AutoMapper)
* Fluent Validation
* TCP/IP ve ComPort Haberleşme
* MQTT + JWT
* TimeOut Service (Redis)
